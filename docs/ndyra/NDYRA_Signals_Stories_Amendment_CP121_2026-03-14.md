# NDYRA Signals / Stories Amendment - CP121

## Rule
- **Signals** = alerts / prompts / notification layer
- **Stories** = content

## Why this matters
The product was drifting toward using “Signals” like content. That creates confusion. The new rule keeps alerts separate from social browsing.

## Core impact
- `/app/signals/` is now aligned to alerts/prompts language.
- `/app/stories/` is reserved for story content and social sharing.
- Profile, Aftermath, and For You can point into Stories without relabeling them as Signals.
