# NDYRA Live Execution — CP106

CP106 adds a dedicated Admin Execute page (`/admin/execute/`) plus runtime readiness banners across billing/marketplace member surfaces.

## What it does
- Makes local preview vs deploy/runtime state obvious
- Prevents false-confidence marketplace/billing flows in half-wired environments
- Adds a copyable step-by-step execution checklist for local, staging, and production

## Surfaces touched
- /admin/execute/
- /admin/status/
- /admin/wiring/
- /app/account/
- /biz/account/
- /app/shop/
- /app/wallet/
- /app/purchases/
- /app/library/timers/

## Intent
CP106 is the first checkpoint that treats deployment execution as a product surface, not just documentation.
